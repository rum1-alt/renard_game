# Renard Game

棋类对战平台 GUI 版。

## 运行

```bash
python3 main.py
```

当前已实现内容：

- Tkinter 图形窗口
- 五子棋 / 围棋模式选择
- 8 到 19 路棋盘大小设置
- 鼠标点击棋盘落子
- 当前行棋方、胜负状态、操作反馈显示
- 悔棋、虚着、认输、重新开始
- 通过文件选择窗口保存和读取局面
- 五子棋连五/平局判断，围棋提子、自杀手、打劫重复局面和无合法落点判断
- 浏览器 GUI 支持重新开始当前局和棋盘星位显示

## 测试

```bash
python3 -m unittest discover -s tests -v
```
