# Renard Game

棋类对战平台浏览器 GUI 版。

## 运行

```bash
python3 main.py
```

当前已实现内容：

- 浏览器图形界面
- 五子棋 / 围棋模式选择
- 默认 15 路棋盘
- 鼠标点击棋盘落子
- 当前行棋方、胜负状态、操作反馈显示
- 悔棋、虚着、认输、重新开始
- 浏览器导出和导入局面
- 五子棋连五/平局判断，围棋提子、自杀手、打劫重复局面和无合法落点判断
- 浏览器 GUI 支持重新开始当前局和棋盘星位显示

## 测试

```bash
python3 -m unittest discover -s tests -v
```
